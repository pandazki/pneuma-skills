Co-creation infrastructure

Visual environment, skills, continuous learning, and distribution — everything humans and code agents need to build content together. Built atop Claude Code, Pneuma gives both of you a shared workspace to think in.

Four pillars of isomorphic collaboration

Not a chat wrapper. Not a code generator. A shared medium where human intent and agent capability meet.

01 — See

Visual Environment

The agent edits files on disk. You see, select, and guide the rendered result in a live, bidirectional workspace. Both operate on the same representation.

02 — Know

Skills

Domain knowledge and seed templates injected per mode. A presentation skill teaches rhythm and export. A document skill teaches prose. Skills version and evolve.

03 — Learn

Continuous Learning

The Evolution Agent mines cross-session history to extract your preferences, then augments the skill with learned patterns. Your agent gets better at being your agent.

04 — Share

Distribution

Build a custom mode with AI via Mode Maker, publish to the marketplace, and let anyone install it instantly. A complete ecosystem for sharing capabilities.

Built-in modes

Each mode is a complete vertical — skill prompt, live viewer, seed templates, and export.

doc Markdown editing with live rendered preview
slide HTML presentations — content sets, drag-reorder, presenter mode, PDF/image export
draw Excalidraw whiteboard with .excalidraw file editing
webcraft Live web development with Impeccable AI design intelligence — 17 commands, responsive preview, export
mode-maker Create custom modes with AI assistance — fork, play-test, publish
evolve Evolution Agent — analyze history, propose skill improvements, apply or rollback
