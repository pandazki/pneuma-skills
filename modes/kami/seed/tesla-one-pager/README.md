# Tesla One-Pager (CN demo)

Adapted from [tw93/kami](https://github.com/tw93/kami) README demo
`assets/demos/demo-tesla.html` (MIT). Stylesheet + asset paths rewritten
to use the kami-mode shared stylesheet at `../_shared/styles.css`.

This is a worked example of a single-page Chinese company one-pager. Edit
the content inside `<div class="page">…</div>`; leave the tokens in
`../_shared/styles.css` alone unless you know what you're changing.
