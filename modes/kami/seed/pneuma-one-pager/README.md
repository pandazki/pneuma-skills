# Pneuma One-Pager (EN demo)

Single-page A4 product brief. Uses the kami one-pager layout originally
contributed by [tw93/kami](https://github.com/tw93/kami) (MIT) — header +
metrics row + lead paragraph + two-column deep-dive + timeline +
milestones + callout quote.

Content is about Pneuma itself: what it is, the four pillars, the
principles, the arc of releases, the current mode roster. Edit the copy
inside `<div class="page">…</div>` to retarget for your own product brief.
Leave the tokens in `../_shared/styles.css` alone unless you know what
you're changing.

Strict constraint: the page must still fit on a single sheet after edits.
